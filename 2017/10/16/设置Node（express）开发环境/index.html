4. 使用nodemon实现服务自动重启

到目前为止，每次你对Express网站进行更改时，必须停止并重新启动服务器才能生效，这很快会让人感到恼火。因此，我们需要借助一个工具在文件有更改时可以自动重启服务。

现在，如果您在项目中编辑任何文件，服务器将重新启动(或者您可以在任何时候通过在命令行上键入rs来重启)。不过，您仍然需要重新加载浏览器以刷新页面。

注意:

我们现在必须调用”npm run <scriptname>“，而不是”npm start“，因为“start”实际上被映射到指定的脚本。我们也可以在“start”脚本中替换这个命令，但是我们只希望在开发期间使用nodemon，所以创建一个新的脚本命令是有意义的。
