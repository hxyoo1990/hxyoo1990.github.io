发表于 2017-10-16

为了使用Express，您首先必须在操作系统上安装Nodejs和Node Package Manager（NPM）。以下部分介绍在Ubuntu Linux 16.04，Mac OS X和Windows 10上安装NodeJs的长期支持（LTS）版本的最简单方法。

1.安装Node

在Windows和Mac OS X上安装Node和NPM是直截了当的，因为您只需使用提供的安装程序：

转到https://nodejs.org/en/

通过双击下载的文件并按照安装提示安装node（node中自带NPM）。
